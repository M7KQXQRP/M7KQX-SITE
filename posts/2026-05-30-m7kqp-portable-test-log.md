Testing out the new home station ground system configuration tonight. Hooked up the multi-strand 16-gauge copper wire to the main grounding rod outside and the noise floor dropped a noticeable couple of S-units on 20m. 

### Digital Mode Setup
Fired up the **Xiegu G90** via the DE-19 interface on macOS Tahoe. Audio input maps cleanly inside the SDR Console configuration setup now. 

* **Power Output:** 20W QRP
* **Mode:** data-U (FT8 tracking)
* **Antenna:** End-Fed Random Wire

Signals are tracking solidly across the waterfall display. Will update with a full performance review once I log a few more contacts down into Europe.
